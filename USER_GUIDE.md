# University of Cape Coast Financial Management System — User Guide

📖 **The full, illustrated user guide is [`USER_GUIDE.html`](USER_GUIDE.html).**

- Open it in any browser (double-click the file), or in the app via the sidebar
  **Finance Suite → 📖 User Guide**, or at the URL **`/USER_GUIDE.html`**.
- Press **Ctrl/Cmd + P** to save it as a PDF for circulation.

It covers every module — Projects & Grants, Budgets, Commitments, Payments,
Journal Vouchers, Procurement (PO → GRN → 3-way match), Accounts Receivable &
Payable, Batch settlement, Recurring entries, Stores & reorder, Fixed Assets,
Payroll, Cash Forecast, Working Capital, Month-End Flash, Trends, Statutory Tax
Pack, Debtor Reminders, Bank Reconciliation, Reports/Export/Import — and **Mo**,
the AI finance assistant — with diagrams, how-to steps and a month-end close playbook.
